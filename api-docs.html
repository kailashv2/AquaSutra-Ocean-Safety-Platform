AquaSutra API Documentation

Comprehensive API documentation for integrating with AquaSutra's ocean safety platform. Access real-time hazard data, submit reports, and build powerful ocean safety applications.

API Endpoints

  • Authentication
  • Hazard Data
  • Submit Reports
  • Monitoring
  • Emergency
  • Analytics
  • Webhooks

POST Authentication

https://api.aquasutra.com/v1/auth/token

Obtain an API access token for authenticated requests. All API calls require a valid Bearer token.

Request Body

{ "api_key": "your_api_key_here", "secret": "your_secret_key_here" }

Response

{ "access_token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...", "token_type": "Bearer", "expires_in": 3600 }

GET Get Hazard Data

https://api.aquasutra.com/v1/hazards

Retrieve real-time ocean hazard data including location, severity, and type information.

Query Parameters

Parameter Type Required Description
lat float Optional Latitude for location-based filtering
lng float Optional Longitude for location-based filtering
radius integer Optional Search radius in kilometers (default: 50)
severity string Optional Filter by severity: low, medium, high

Response

{ "hazards": [ { "id": "HAZ-001", "type": "high_waves", "severity": "high", "location": { "lat": 40.7128, "lng": -74.0060, "name": "New York Harbor" }, "description": "Wave heights exceeding 3.5 meters", "timestamp": "2024-01-15T10:30:00Z", "active": true } ], "total": 1, "page": 1 }

POST Submit Hazard Report

https://api.aquasutra.com/v1/reports

Submit a new hazard report to the AquaSutra platform for review and processing.

Request Body

{ "hazard_type": "high_waves", "severity": "high", "location": { "lat": 40.7128, "lng": -74.0060, "description": "Miami Beach, FL" }, "description": "Dangerous wave conditions observed", "reporter": { "name": "John Doe", "email": "john@example.com", "phone": "+1234567890" } }

Response

{ "report_id": "RPT-001", "status": "submitted", "message": "Report submitted successfully", "estimated_review_time": "15 minutes" }

GET Monitoring Data

https://api.aquasutra.com/v1/monitoring/sensors

Access real-time data from ocean monitoring sensors including temperature, wave height, and water quality.

Response

{ "sensors": [ { "id": "NYC-001", "name": "New York Harbor Buoy", "location": { "lat": 40.7128, "lng": -74.0060 }, "status": "online", "readings": { "temperature": 22.5, "wave_height": 1.8, "wind_speed": 12, "water_quality": 88 }, "last_update": "2024-01-15T10:30:00Z" } ] }

POST Emergency Alert

https://api.aquasutra.com/v1/emergency/alert

Submit an emergency alert for immediate response coordination.

Request Body

{ "emergency_type": "person_in_water", "priority": "critical", "location": { "lat": 40.7128, "lng": -74.0060, "description": "Miami Beach, FL" }, "description": "Person in distress in water", "contact": { "phone": "+1234567890", "name": "Emergency Caller" } }
